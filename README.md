Bank System Improved - Sistem bancar avansat în Java cu suport multi-valută (MDL, EUR, USD, GBP, RON). 
Include gestionare conturi, operațiuni bancare (depuneri, retrageri, transferuri), 
schimb valutar cu rate dinamice, rapoarte detaliate și sistem de securitate cu 
autentificare duală (client/manager). Proiect modular cu interfețe console intuitive, validări riguroase 
și gestionare completă a tranzacțiilor. Ideal pentru demonstrarea competențelor în 
OOP, Collections și logică de business complexă.
